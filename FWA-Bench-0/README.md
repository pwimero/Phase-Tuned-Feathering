# FWA-Bench-0

Version: 0

Feathered Wingtip Analytical Benchmark for the theory-only radiating-covariance paper. The package contains canonical diagonal, rank-one, controlled-mechanism, and partial-coherence cases plus expected outputs generated by `scripts/run_radiating_covariance_validation.py`.

The benchmark is not CFD or wind-tunnel validation. It verifies exact algebra, canonical coherent-array behavior, diagonal no-phase behavior, controlled mechanism examples, partial-coherence transition behavior, and the feathered-wingtip campaign diagnostics used in the manuscript.
